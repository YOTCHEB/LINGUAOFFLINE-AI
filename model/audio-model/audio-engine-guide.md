# Audio Engine Guide (starter)

Recommended offline engines:

- ASR: Vosk (https://alphacephei.com/vosk/)
- TTS: Coqui TTS (https://github.com/coqui-ai/TTS)

Instructions:
- Install Vosk and download relevant language models.
- Install Coqui-TTS and select a lightweight voice model.
- Use example scripts (ASR/TTS) to test locally.
